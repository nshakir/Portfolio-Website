CLIENT

Push/Pull Studios

Push Pull Studio is a recording studio based in Seattle, WA. One of the client's concern was to be able to use SEO to have his website be more searchable. Push Pull Studio was the first time I built a customized theme through WordPress. I used content audit, information architectures, visual and functional design to get this website off the ground.

Push/Pull Studio wanted the website to be clean and simple, but for the user to be interactive. SoundCloud music player was installed on the website and we are working on having a live video feed be installed so that the user can watch how the studio works with their clients

A new thing that I learned from this project was the difference between Wordpress.com and Wordpress.org and how Wordpress.com accounts does not let you do a quick install for customizable themes as Wordpress.org would. With this issue, I did redirects of Push Pull domain from their original Wordpress.com account to a new Wordpress.org account in order to install a customized theme.

USER PERSONAS

The Band

This would be the artist or band that would be seeking Push/Pull for recording studio purposes. They would more than likely be local, seeking smaller studios that are affordable. They would seek pricing, basic information such as location and a bio about the owner of Push/Pull, photos of the studio, the equipment they have to offer and how to contact the studio.

The Collaborator

Another recording studio owner, or audio engineer seeking Push/Pull for potential collaborations in the future. Local audio engineer with a project seeking another source. They would seek the credentials for the owner of Push/Pull, pricing, contact information and equipment the studio has to offer.
