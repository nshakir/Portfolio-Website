E-Commerce UX Case Study

Mendenhall & Rodden

Mendenhall & Rodden involved creating a fictional women hiking apparel online retailer to study the elements of user experience, from developing business goals to the customer checkout process. The process included defining site objectives and understanding user needs, creating an effective information architecture, building wireframes, designing visual design, and hand-coding a click through protosite.
2015

SCOPE
  • Web Design
  • UX Design

UX ANALYSIS

User Experience was needed for the site. The client goal was for the user to go to the website for information and as a reference guide. We gathered information for a content audit, implementing a sitemap, and designing a set wireframes for the entire site.
